# STYLE GUIDE WIX — NFC YOUCAR

## Direction artistique
Page mobile-first, très blanche, premium, sobre, proche du site YouCar.

## Couleurs
Noir : #0B0B0B  
Rouge YouCar : #D9142B  
Gris clair : #F3F3F3  
Gris ligne : #D9D9D9  
Texte secondaire : #777777  
Blanc : #FFFFFF  

## Typographies
Utiliser une police sobre proche du site :
- Helvetica / Arial / Sans-serif
- Titres en majuscules
- Letter spacing important sur les titres

## Mise en page
Largeur mobile prioritaire.
Beaucoup d’espaces blancs.
Logo centré en haut.
Traits fins noirs ou gris pour structurer.
Rouge uniquement en accent : petits traits, sous-titres, flèches, étoiles.

## Boutons
Boutons blancs, bordure gris clair, ombre très légère.
Texte noir.
Flèche rouge à droite.
Ne pas utiliser de gros boutons rouges.

## Sections
Titre en majuscules espacées :
NOS SERVICES
SUIVEZ-MOI
AVIS CLIENTS

Ajouter un petit filet rouge sous chaque titre.

## Image
Image Porsche sobre, premium, claire.
Éviter les visuels trop commerciaux.
Préférer une Porsche de profil, grise/noire, ambiance catalogue.
