# GUIDE NFC TOOLS — CARTE YOUCAR

## Objectif
La carte NFC doit ouvrir la page :

https://www.you-car.fr/cyrille

## Étapes dans NFC Tools

1. Ouvrir NFC Tools
2. Aller dans l’onglet Écrire
3. Ajouter un enregistrement
4. Choisir URL / URI
5. Coller :
   https://www.you-car.fr/cyrille
6. Valider
7. Cliquer sur Écrire
8. Poser le téléphone sur la carte NFC
9. Attendre le message d’écriture réussie

## Test
Fermer NFC Tools.
Approcher un iPhone ou Android compatible NFC de la carte.
Le téléphone doit proposer d’ouvrir www.you-car.fr/cyrille.

## Recommandation
Ne pas écrire plusieurs liens sur la carte.
Toujours écrire uniquement l’URL de la page YouCar.
Si les liens changent, modifier la page Wix sans reprogrammer la carte.
